## 📋 口号使用指南 | Slogan Usage Guidelines

### ✅ 正确使用规范 | Proper Usage Standards

#### 场合适配原则 | Occasion Adaptation Principles
```
正式场合: 使用专业、稳重的口号
技术交流: 突出技术优势和创新点
用户推广: 强调易用性和价值收益
媒体报道: 使用有新闻价值的表述
```

#### 语言风格指南 | Language Style Guide
```
中文风格: 简洁明了，富有文化内涵
英文风格: 直接有力，符合国际习惯
技术表述: 准确专业，避免过度夸张
情感表达: 真诚自然，建立情感连接
```

### ❌ 避免使用情况 | Usage to Avoid

#### 禁忌表述 | Taboo Expressions
```
避免绝对化: "最好的"、"唯一的"
避免夸张: "革命性的"、"颠覆性的"
避免贬低竞品: "比其他更好"
避免技术误导: 夸大技术能力
```

#### 文化敏感性 | Cultural Sensitivity
```
避免文化冲突表述
注意多语言文化差异
尊重各地区习俗传统
确保全球通用性
```
