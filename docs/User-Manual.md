# 用户手册

本文档提供灵犀/MindSync的完整功能使用说明，帮助您充分利用这款跨平台剪贴板同步工具。

## 🎯 功能概览

### 核心功能
- **实时同步** - 跨设备剪贴板内容实时同步
- **历史管理** - 剪贴板历史记录查看和管理
- **设备管理** - 多设备连接状态监控
- **安全加密** - 端到端数据加密保护

### 高级功能
- **选择性同步** - 按内容类型和设备选择性同步
- **快捷操作** - 全局快捷键和快速菜单
- **备份恢复** - 数据备份和恢复功能
- **自定义规则** - 同步规则自定义配置

## 🚀 基础使用

### 首次设置

#### 1. 账号注册与登录
1. 打开灵犀应用
2. 点击"注册新账号"
3. 输入邮箱和设置密码
4. 完成邮箱验证
5. 登录账号

#### 2. 设备连接
1. 在其他设备上安装灵犀应用
2. 使用同一账号登录
3. 系统自动发现并连接设备
4. 确认设备授权

#### 3. 基础同步测试
1. 在设备A复制文本：`Hello MindSync!`
2. 在设备B粘贴验证同步
3. 测试图片同步功能

### 界面介绍

#### 系统托盘图标（桌面端）
- **绿色** - 连接正常
- **黄色** - 连接中/部分连接
- **红色** - 连接断开
- **灰色** - 应用未运行

#### 主界面功能区域
1. **设备列表** - 显示所有连接设备
2. **同步状态** - 实时同步状态显示
3. **历史记录** - 剪贴板历史管理
4. **设置面板** - 功能配置区域

## 🔧 功能详解

### 剪贴板同步

#### 同步内容类型
- **文本内容** - 纯文本、富文本
- **图片内容** - PNG、JPEG、GIF等格式
- **文件路径** - 本地文件路径同步（可选）
- **自定义数据** - 应用特定格式数据

#### 同步策略配置

**实时同步模式**
- 立即同步所有剪贴内容
- 最低延迟，最高实时性
- 适合对实时性要求高的场景

**手动同步模式**
- 用户手动触发同步
- 节省网络流量
- 适合移动网络环境

**定时同步模式**
- 按设定时间间隔同步
- 平衡实时性和资源消耗
- 可配置同步频率（1分钟-1小时）

### 历史记录管理

#### 历史查看
- 时间顺序排列
- 内容预览功能
- 搜索和过滤
- 批量操作支持

#### 历史操作
- **复制** - 重新复制历史内容
- **删除** - 删除单条或批量删除
- **收藏** - 标记重要内容
- **导出** - 导出历史记录

#### 自动清理设置
- 按时间自动清理（1天-1年）
- 按数量自动清理（100-1000条）
- 排除收藏内容
- 手动清理选项

### 设备管理

#### 设备状态监控
- 在线/离线状态
- 最后活动时间
- 同步统计信息
- 网络连接质量

#### 设备操作
- **重命名设备** - 自定义设备名称
- **断开连接** - 临时断开设备
- **移除设备** - 永久移除设备
- **传输统计** - 查看数据传输量

#### 设备分组
- 按用途分组（工作/个人）
- 按位置分组（家庭/办公室）
- 按平台分组（桌面/移动）
- 自定义分组规则

## ⚙️ 高级配置

### 同步规则定制

#### 内容过滤规则
```yaml
# 示例配置
rules:
  - type: text
    conditions:
      - min_length: 10      # 最小长度
      - max_length: 1000    # 最大长度
      - exclude_patterns:   # 排除模式
        - "密码"
        - "密钥"
    action: sync
    
  - type: image
    conditions:
      - max_size: 5MB       # 最大文件大小
      - formats: [png, jpg] # 支持格式
    action: sync
```

#### 设备同步策略
- **全设备同步** - 所有设备都同步
- **选择性同步** - 指定设备同步
- **排除同步** - 排除特定设备
- **条件同步** - 基于网络条件同步
