## 构建系统配置

### CMake配置
```cmake
# 核心配置
set(PROJECT_NAME "cross-platform-ime")
set(VERSION_MAJOR 1)
set(VERSION_MINOR 0)
set(VERSION_PATCH 0)

# 平台检测
if(WIN32)
    set(PLATFORM_WINDOWS TRUE)
elseif(APPLE)
    set(PLATFORM_MACOS TRUE)
elseif(UNIX AND NOT APPLE)
    set(PLATFORM_LINUX TRUE)
endif()

# 模块化构建选项
option(BUILD_CORE "Build core IME engine" ON)
option(BUILD_PLUGINS "Build plugin system" ON)
option(BUILD_CLOUD "Build cloud services" OFF)
option(BUILD_TESTS "Build tests" ON)

# 依赖查找
find_package(Threads REQUIRED)
```

## 开发环境设置

### 依赖要求
- **C++17** 兼容编译器
- **CMake 3.15+**
- **平台特定SDK**
  - Windows: Windows SDK 10.0+
  - macOS: Xcode 12.0+
  - Linux: IBus/Fcitx开发包
  - Android: NDK 21+
  - iOS: Xcode 12.0+
  - HarmonyOS: DevEco Studio 3.0+

### 构建命令
```bash
# 配置项目
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release

# 构建所有目标
cmake --build . --config Release

# 运行测试
ctest --output-on-failure
```
